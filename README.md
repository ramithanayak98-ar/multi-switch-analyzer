Multi Switch Analyzer
Overview:
This project is a simple Python-based tool to analyze network topologies with multiple switches. It helps in understanding how switches and devices are connected and checks whether the topology is valid or not.

Features:
Shows how switches and devices are connected
Performs basic validation on the network
Easy to understand and modify
Useful for learning networking concepts

Project Files:
multi-switch-analyzer/
│── topology.py
│── test_validation.py
│── screenshots/
│── README.md

How to Run:
Clone the repository
git clone https://github.com/ramithanayak98-ar/multi-switch-analyzer.git
Go to the folder
cd multi-switch-analyzer
Run the file
python topology.py

Output:
Screenshot folder

Where this can be used:
-For understanding basic network topology
-For college mini projects
-For practicing Python + networking concepts
